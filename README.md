# Basic-Data-Cleaning--Indian-Ecommerce--SQL

## Objective 
The objetive of this project to clean the dataset with using basic SQL function.

## Dataset issue identification:
The dataset having certain issue which is given below-
- Duplicate Data
- Null Values
- Invalid Formatt
- Extra spaces in columns
- Spelling mistake
- Multiple Date format
- Negative Number

# Tool use:
- MySQL
- .CSV file
 
## Dataset Cleaning Steps
- Query 1. Find the Duplicates value in dataset
- Query 2. Find the Empty OR null values in dataset
- Query 3. Find Duplicate value in Entire Dataset
- Query 4. Checking issue in Other Columns (Email, Phone, Price, Quantity, Rating)
- Query 5. Bussiness Logic
- Query 6 - 7. Date Formatt in Delivery & Order Date
- Query 8 - 9. Fix Date Formatt from Delivery & Order Date
- Query 10 - 12. Remove Duplicates 
- Query 13 - 15. Replace the Empty values in Delivery & Order Date Columns 
- Query 16 - 19. Clean null value from the Date, Name, Emails, & Phone columns
- Query 20 - 21. Standardize the city & Category
- Query 22. Clean Price
- Query 23 - 24. Fix the Quantity & Rating
- Query 25. Change the data type
